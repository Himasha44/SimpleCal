🧮 Simple Calculator

A simple and responsive calculator built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations with a clean and user-friendly interface.

📌 About The Project

This calculator was built to practice JavaScript logic and DOM manipulation.
It performs basic mathematical operations and updates results dynamically based on user input.

🛠️ Built With

HTML5 – Structure

CSS3 – Styling

JavaScript (Vanilla JS) – Functionality & Logic

✨ Features

Addition, Subtraction, Multiplication, Division

Clear (AC) button

Real-time display update

Responsive design

Keyboard support (if you added it)
